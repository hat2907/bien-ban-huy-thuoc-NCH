💾 Lưu biên bản vào Google Sheet (Apps Script)

Mỗi lần nhân viên bấm "Lưu biên bản", dữ liệu sẽ được ghi tự động vào một Google Sheet tổng hợp. Cần thiết lập một lần qua Google Apps Script (miễn phí).
⚪ Chưa cấu hình
  1. Tạo một Google Sheet mới để chứa dữ liệu biên bản (đặt tên tuỳ ý, ví dụ: Tổng hợp biên bản hủy 2026).
  2. Mở sheet đó → bấm Extensions → Apps Script.
  3. Xoá code mặc định, dán đoạn code sau vào rồi bấm Save (Ctrl+S):
function doPost(e) {
  try {
    var rows = JSON.parse(e.postData.contents);
    var sheet = SpreadsheetApp.getActiveSpreadsheet().getActiveSheet();
    rows.forEach(function(row){ sheet.appendRow(row); });
    return ContentService
      .createTextOutput(JSON.stringify({status:'ok',saved:rows.length}))
      .setMimeType(ContentService.MimeType.JSON);
  } catch(err) {
    return ContentService
      .createTextOutput(JSON.stringify({status:'error',msg:err.toString()}))
      .setMimeType(ContentService.MimeType.JSON);
  }
}
  1. Bấm Deploy → New deployment → Web app.
    Chọn: Execute as: Me | Who has access: Anyone → Deploy.
    Lần đầu Google sẽ yêu cầu cấp quyền → chọn tài khoản → Advanced → Go to … → Allow.
  2. Copy Web app URL (dạng https://script.google.com/macros/s/…/exec) và dán vào ô trên.

⚠️ Mỗi khi sửa code, cần deploy lại (New deployment hoặc New version) để thay đổi có hiệu lực.

2. Tiến hành tiêu hủy

Căn cứ theo đề xuất hủy của khoa, Bệnh viện tiến hành hủy bao bì tiếp xúc trực tiếp và dư phẩm thuốc gây nghiện, thuốc hướng thần và tiền chất với các nội dung như sau:

Phương pháp hủy:

- Với bao bì TXTT: Phá hủy bằng cách cắt/ đập vỡ/nghiền biến dạng rồi xử lý rác thải theo qui định của Bệnh viện

- Với dư phẩm thuốc: Đổ dư phẩm thuốc vào xô nước để hòa loãng rồi đổ vào đường nước thải chung

Thành phần tham gia:

1.Khoa Dược, Thành viên Hội đồng hủy
2.Phòng TCKT, Thành viên Hội đồng hủy
Khoa DP&KSNK, Thành viên hủy
3.Khoa Dược
4.Khoa DP&KSNK

Chúng tôi cùng chứng kiến và thực hiện hủy hết lượng bao bì TXTT và dư phẩm thuốc gây nghiện, hướng thần và tiền chất đúng như trên Mục 1.

Việc tiêu hủy kết thúc vào lúc giờ phút cùng ngày.
